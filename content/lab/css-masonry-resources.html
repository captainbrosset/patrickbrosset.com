Try the feature

To try the feature in your own projects:

  • Open a Chromium-based browser, such as Microsoft Edge or Chrome.
    The feature is currently only implemented in these browsers.
  • In the browser, open a new tab and go to about:flags.
  • In the Search flags box, enter CSS Masonry Layout.
  • Set the CSS Masonry Layout flag to Enabled.
  • Click the Restart button.

Share feedback

To share feedback about the design of the feature, please open an issue on the CSS Working Group repository.

To report a bug about the implementation of the feature in Chromium instead, please open a bug on the Chromium bug tracker.
