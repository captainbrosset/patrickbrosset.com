Try the feature

To try the feature in your own projects:

  • Open a Chromium-based browser, such as Microsoft Edge or Chrome, and make sure the version is at least 139.
    The feature is currently only implemented in these browsers.
  • In the browser, open a new tab and go to about:flags.
  • In the Search flags box, enter Experimental Web Platform features.
  • Set the Experimental Web Platform features flag to Enabled.
  • Click the Restart button.
