Today is the last day of 2025, which seems like a good time to look back and think about what I've done this year.
This year, more than ever, I've continued to push for a web that's easier to build with, more capable, easier to discover, and better documented.
I'm thankful to be able to do this work, something I enjoy and care deeply about.

## I documented web technologies and made them easier to find and use

* I created over 180 commits to [the web-features repository](https://github.com/web-platform-dx/web-features), to help build a better, more comprehensive, catalog of all the features of the web platform.

* I created about 10 commits to the [mdn/content](https://github.com/mdn/content) and [mdn/browser-compat-data](https://github.com/mdn/browser-compat-data/) repositories, to improve MDN web docs and browser compatibility tables.

* I created over 140 commits to the [MicrosoftDocs/edge-developer repository](https://github.com/MicrosoftDocs/edge-developer), to improve the [web platform-related Edge technical documentation](https://learn.microsoft.com/microsoft-edge/developer/), in particular publishing [Edge Beta release notes](https://learn.microsoft.com/microsoft-edge/web-platform/release-notes/) every 4 weeks.

* I created about 200 commits both on the [web-features-mappings](https://github.com/web-platform-dx/web-features-mappings) and [web-features-explorer](https://github.com/web-platform-dx/web-features-explorer) repos, which together power [the Web Platform Features Explorer site](https://web-platform-dx.github.io/web-features-explorer/), a tool to discover and learn about the web platform, via [monthly release notes](https://web-platform-dx.github.io/web-features-explorer/release-notes/december-2025/), and [detailed feature pages](https://web-platform-dx.github.io/web-features-explorer/features/document-caretpositionfrompoint/).
