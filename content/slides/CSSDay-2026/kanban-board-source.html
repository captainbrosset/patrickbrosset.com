.kanban-board {
  display: grid-lanes;
  grid-template-columns: 1fr repeat(2, 1.5fr) 1fr;
  gap: 1rem 2rem;
}

[data-column="backlog"] { grid-column: 1; }
[data-column="doing"]   { grid-column: 2; }
[data-column="review"]  { grid-column: 3; }
[data-column="done"]    { grid-column: 4; }
Again, display:grid-lanes with columns. Here columns have different widths, just for fun. And each item has a data-column attribute which determines which lane it belongs to. So the code doesn't change the grid-column property directly. It changes the data attribute.
