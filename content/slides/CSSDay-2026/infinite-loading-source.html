.infinite {
  display: grid-lanes;
  grid-template-columns: repeat(auto-fill, minmax(40vmin, 1fr));
  gap: 3vmin;
}

.item.loading {
  grid-column: 1 / -1;
}
// Load more when the last card intersects the viewport.
const observer = new IntersectionObserver(async entries => {
  if (entries.some(e => e.isIntersecting)) {
    const items = await Promise.all(Array.from({ length: BULK_LOAD_SIZE }, () => fetchAndCreateItem()));
    items.forEach(item => container.appendChild(item));
    observer.observe(container.lastChild);
  }
}, { threshold: 0.25 });
Simple display:grid-lanes, with some auto-repeating columns and some gap. Items have nothing special except the last one that's used for the loading indicator, and that spans all columns. And then, over in JS, we have our IntersectionObserver
