And for buttons like these, that have hover and active states, you'd have to make custom images, for each button and each button state (normal, hover, active), and include the text in the images, if you wanted a fancy font. And then handle the swapping of images in JavaScript. I personally used Dreamweaver because it had a cool image rollover feature that wrote the code for me. Again, images were the name of the game. We'd often create sites from images, slice them up into little pieces to achieve the desired design. And then position everything with absolute pixel values, using tables to make the end result look exactly like the initial image design.
