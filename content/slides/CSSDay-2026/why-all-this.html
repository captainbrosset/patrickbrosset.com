--- layout: css-day-slides.njk title: Why all this? notes: | First, because computers are fun. And that's probably enough of a reason. But also, wearing my browser vendor hat on for a minute: this is how the web evolves. If the web became that much more capable, do we still need to play and explore? Well, if we don't play with the features of the web platform, actual html, css, js features, we end up pushing the platform forward far less and that's bad for everybody. For our beloved platform to remain capable and interoperable we, all together as a community, need to be constantly filling the pipe with new features so that these features can then become standardized and then interoperable based on actual usage and real feedback. There's risk in simply copy/pasting from old solutions Using AI that doesn't know about the latest features, or using only framework-provided solutions which abstract primitives away A platform that stagnates is not a great platform. We need to keep playing with the web, explore its edges, and even go beyond them. ---

Why all this?

Because computers are fun.

Oh, and also because that's how the web remains relevant.
